•Iconografía•

Con respecto a los iconos, no se busca el uso de iconos muy elaborados o recargados, sino mas bien el uso de iconografía simple pero colorida.
Aunque se use iconografía hecha con vectores se dará prioridad a iconografía mas minimalista y colorida, dando un toque más desenfadado que formal.
Ejemplo de diversos iconos que se podrían usar en la página web. Para búsquedas, carrito de la compra con los productos elegidos, para diferenciar las categorías de juguetes... Todos siguen el mismo principio de diseño: Diseños coloridos de estilo infantil, con líneas gruesas para facilitar su reconocimiento.

¡Bienvenido/a a la página web de Toys'R'Us!

Esta página web es un proyecto realizado para mostrar la importancia de una buena guía de estilos.
Una guia de estilos se define como un conjunto de reglas y recomendaciones que establecen cómo se debe diseñar y estructurar el código HTML y CSS de un sitio web para garantizar la coherencia, la legibilidad y la facilidad de mantenimiento del código.
Es la biblia de la parte visual de nuestra página web. Por eso es un paso vital en la creación de cualquier página de internet. Nos ayudará a mantener la coherencia visual -ya sea una web simple con un par de enlaces, o una web con cientos de lugares que visitar-, mediante el uso de colores, iconos, imágenes, tipografías, formas de escritura, etc.

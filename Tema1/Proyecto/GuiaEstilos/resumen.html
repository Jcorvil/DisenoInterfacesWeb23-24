•Resumen•



-Público Objetivo-

El público objetivo de nuestra página web son principalmente niños, aunque también es una página dirigida a los padres, tíos, abuelos o cualquier familiar interesado en comprar juguetes. El objetivo es ofrecer una web fácil de navegar y atractiva tanto a ojos de los niños como de los padres, atrayendo a los niños a los diversos productos en los que podrían estar interesados, a la vez que se le recomiendan otros productos a los padres.



-Tipo de web-

La web será una web de compras online. Se dispondrá de varios catálogos para recoger los distintos tipos de juguetes, agrupados según edad, gustos de los posibles compradores, tipo de juego, etc.
Estos catálogos se podrán modificar con el tiempo para mostrar nuevos juguetes, ofertas u ocasiones especiales como navidades.
